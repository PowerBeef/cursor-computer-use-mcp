# Benchmarks

Headless MCP harness for native `open-computer-use mcp` (OCU). Optional legacy comparison against `npx computer-use-mcp`.

**Documentation:** [docs/BENCHMARK.md](../docs/BENCHMARK.md)

```bash
BENCHMARK_TRIALS=1 npm run benchmark
```

Do **not** commit `benchmarks/results/` — local reports only (see repo `.gitignore`).

Manual Composer eval: [composer-prompts.md](composer-prompts.md), [composer-rubric.md](composer-rubric.md), templates `templates/composer-ocu.csv` and `templates/composer-legacy.csv`.
